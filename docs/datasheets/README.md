# Hardware Datasheets

The board in the supplied photos appears to be an ESP32-C3 Super Mini development board. The rear silkscreen says `SuperMini`, and the front side exposes ESP32-C3 GPIO labels such as `20`, `21`, `8`, `9`, `3V3`, `5V`, `G`, `RST`, and `BOOT`.

- `espressif-esp32-c3-datasheet.pdf` - ESP32-C3 SoC datasheet from Espressif.
- `espressif-esp32-c3-technical-reference-manual.pdf` - ESP32-C3 technical reference manual from Espressif.
- `espressif-esp32-c3-hardware-design-guidelines.pdf` - ESP32-C3 hardware design guidelines from Espressif.
